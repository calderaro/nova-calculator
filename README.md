Nova Calculator
===============

## Installation

### Clone
```bash
git clone git@github.com:calderaro/nova-calculator.git
```
### Install dependencies
```bash
npm install
```
### Start server
```bash
node index
```

### Enter
```bash
http://localhost:3006/
```
